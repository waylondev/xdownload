## Installation

Download the appropriate installer for your platform:

- **Windows**: `xdownload_0.1.0_x64-setup.exe`
- **macOS**: `xdownload_0.1.0_x64.dmg`
- **Linux**: `xdownload_0.1.0_amd64.AppImage`

## System Requirements

- **Windows**: Windows 10 or later
- **macOS**: macOS 10.15 or later
- **Linux**: Ubuntu 18.04+, Fedora 30+, or similar

## Prerequisites

Before using XDownload, ensure you have [yt-dlp](https://github.com/yt-dlp/yt-dlp) installed on your system.

## Usage

1. Launch XDownload
2. Enter download command or use quick commands
3. Files are saved to `xdownloads` folder in current directory

## Troubleshooting

For detailed usage instructions, refer to the [yt-dlp documentation](https://github.com/yt-dlp/yt-dlp).
